# Procedural Design Experiments

A collection of procedural generation experiments focusing on organic forms and reaction-diffusion patterns.

## Project Structure

```
scratch/
├── notebooks/             # Jupyter notebooks per experiment
├── experiments/          # Python scripts extracted from notebooks
├── meshes/               # Generated 3D model outputs
├── renders/              # Visual outputs and renders
├── shared/               # Common utilities
├── docs/                 # Documentation and notes
├── data/                 # Input files and seeds
└── outputs/              # Final exports
```

## Experiments

### Current Projects
- Gray Scott Vase: Reaction-diffusion patterns applied to vessel forms
- Moss Pole: Organic growth patterns on cylindrical structures
- Coral Brain: Complex branching structures using L-systems

## Setup

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Launch Jupyter notebooks:
```bash
jupyter lab
```

3. Navigate to `notebooks/` to run experiments

## Tools & Technologies

- Python 3.9+
- NumPy/SciPy for computation
- Trimesh for 3D mesh operations
- Matplotlib/Plotly for visualization
- Blender (optional) for advanced rendering

## Documentation

Detailed documentation for each experiment can be found in the `docs/` directory.
